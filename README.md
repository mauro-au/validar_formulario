## рҹ§  Decisiones tГ©cnicas destacadas

### рҹҺһпёҸ AnimaciГіn del dialog con `@starting-style`

La entrada del modal usa CSS nativo вҖ” sin necesidad de togglear clases con JS para el estado de apertura:

```css
dialog {
  transition:
    opacity 0.4s ease,
    filter 0.4s ease,
    transform 0.4s ease;

  &[open] {
    opacity: 1;
    filter: blur(0);
    transform: translateY(0);
  }

  @starting-style {
    &[open] {
      opacity: 0;
      filter: blur(7px);
      transform: translateY(-5%);
    }
  }
}
```

La salida aГәn usa una clase `.closing` + `transitionend` por seguridad cross-browser вҖ” el soporte de `overlay allow-discrete` en Firefox es incompleto a 2025.

### рҹҺү Confetti vГӯa Popover API

`canvas-confetti` necesita renderizar en un `<canvas>` por encima del Top Layer del `<dialog>`. EnvolviГ©ndolo en un elemento `popover` lo promueve al Top Layer _despuГ©s_ del dialog, resolviendo el conflicto de apilamiento sin hacks:

```html
<div id="confetti-host" popover>
  <canvas id="confetti-canvas"></canvas>
</div>
```
